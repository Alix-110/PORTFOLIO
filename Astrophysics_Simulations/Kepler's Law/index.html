Kepler's Laws Simulation: Equal Areas in Equal Times

Purpose of the Simulation

This simulation demonstrates Kepler's Second Law: a line connecting a planet to its star sweeps equal areas in equal times. The orbit is elliptical with the star at one focus. The orange shaded area represents the area swept over time, visually confirming the law.

Scientific Background

Kepler's laws describe planetary motion:

  • 1st Law: Planets move in ellipses with the Sun at one focus.
  • 2nd Law: The line joining a planet and the Sun sweeps equal areas in equal time intervals.
  • 3rd Law: The square of the period is proportional to the cube of the semi-major axis.
This simulation focuses on the second law using the formula for the area swept:
dA/dt = constant, where A is the area swept.

Methodology

We approximate the orbit as a parametric ellipse:
x = a cos(θ), y = b sin(θ)
The focus (star) is at (-√(a²-b²), 0).
Using Python libraries:

  • numpy for numerical arrays and parametric calculations.
  • matplotlib for plotting and animation.
At each time step, we:
  1. Calculate planet position.
  2. Draw a polygon from focus to planet path to visualize swept area.
  3. Animate the planet and area to show equal areas in equal times.

Pseudocode


Initialize constants (G, M, a, b, T)
Compute ellipse parameters
Calculate focus coordinates (star position)
Generate time array
Compute parametric positions (x, y) for planet
Setup matplotlib figure and axes
Draw orbit ellipse and star
Create polygon for swept area
Define animation function:
    - Update planet position
    - Update swept area polygon
Run animation using FuncAnimation
Display or save animation

Python Simulation Code


import numpy as np
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation
from matplotlib.patches import Polygon

G = 1.0
M = 1.0
a = 5.0
b = 3.0
T = 20.0
num_points = 500

t = np.linspace(0, T, num_points)
theta = 2 * np.pi * t / T
x = a * np.cos(theta)
y = b * np.sin(theta)
focus_x = -np.sqrt(a**2 - b**2)
focus_y = 0

fig, ax = plt.subplots(figsize=(8, 6))
ax.set_aspect('equal')
ax.set_xlim(-a*1.5, a*1.5)
ax.set_ylim(-a*1.5, a*1.5)
ax.set_title("Kepler's Second Law: Equal Areas in Equal Times")
orbit_line, = ax.plot(x, y, color='gray', linestyle='--', label='Orbit')
ax.plot(focus_x, focus_y, 'yo', markersize=12, label='Star (Focus)')
planet_marker, = ax.plot([], [], 'bo', markersize=8, label='Planet')
swept_area = Polygon([[focus_x, focus_y], [x[0], y[0]]], closed=True, color='orange', alpha=0.3)
ax.add_patch(swept_area)

def animate(i):
    planet_marker.set_data([x[i]], [y[i]])
    swept_area.set_xy(np.vstack([[focus_x, focus_y], np.column_stack((x[:i+1], y[:i+1]))]))
    return planet_marker, swept_area

ani = FuncAnimation(fig, animate, frames=num_points, interval=40, blit=True)
plt.legend()
plt.show()

Code Explanation

Constants and Parameters: Define star mass, orbit size, etc.
Ellipse Parametrization: x = a cos(θ), y = b sin(θ).
Animation: Updates planet position and the swept area polygon frame by frame.
Visualization: Planet as a blue dot, star as yellow dot, area as semi-transparent orange polygon.
