Stellar Evolution Tracks

Animate a star moving across the Hertzsprung–Russell diagram

Purpose

Visualize a star evolving across the H-R diagram from main sequence to red giant using simplified stellar models. The animation demonstrates how stellar temperature decreases while luminosity increases during late evolution.

Background

The Hertzsprung–Russell diagram plots stellar luminosity vs surface temperature. As stars evolve, their surface temperature and luminosity change. Main sequence stars maintain near-constant luminosity, while subgiants and giants increase in luminosity and cool in temperature.

Methodology

  1. Create arrays for temperature and luminosity representing simplified evolutionary stages.
  2. Set up a Matplotlib figure with logarithmic axes and dark background.
  3. Plot the static H-R track in light gray.
  4. Add a moving star marker and trail line for animation.
  5. Use FuncAnimation to animate the star along the track.

Pseudocode


Define arrays for star temperatures and luminosities
Create Matplotlib figure with log-log axes
Plot the static H-R track
Initialize star marker and trail
For each frame i:
    Update star position to temps[i], luminosities[i]
    Update trail to include all previous positions
Render animation with FuncAnimation
Show figure

Python Simulation Code


import numpy as np
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation


plt.style.use("dark_background")


temps = np.concatenate([
    np.linspace(10000, 5800, 50),  # main sequence cooling
    np.linspace(5800, 4000, 50),   # subgiant
    np.linspace(4000, 3500, 50)    # red giant
])

luminosities = np.concatenate([
    np.linspace(1, 1, 50),         # main sequence (constant)
    np.linspace(1, 100, 50),       # subgiant (rising)
    np.linspace(100, 1000, 50)     # red giant (very bright)
])


fig, ax = plt.subplots(figsize=(10, 7))
ax.set_yscale('log')
ax.set_xscale('log')
ax.set_xlim(max(temps), min(temps))  # invert x-axis
ax.set_ylim(min(luminosities)*0.8, max(luminosities)*1.2)
ax.set_xlabel("Temperature (K)")
ax.set_ylabel("Luminosity (L/L_sun)")
ax.set_title("Stellar Evolution Track Animation")

# Plot static track (light gray)
ax.plot(temps, luminosities, color='gray', linestyle='--', alpha=0.5)


# Animated star marker
star_marker, = ax.plot([], [], 'o', color='yellow', markersize=12)

# Trail for star
trail, = ax.plot([], [], color='yellow', alpha=0.5)

# Animation function
def animate(i):
    star_marker.set_data([temps[i]], [luminosities[i]])  # <-- wrap in list
    trail.set_data(temps[:i+1], luminosities[:i+1])
    return star_marker, trail

ani = FuncAnimation(fig, animate, frames=len(temps), interval=100, blit=True)
plt.show()

Explanation

- The temperature array represents the star cooling over time.
- The luminosity array models the star brightening during subgiant and red giant phases.
- A yellow marker represents the star, and a trail shows the previous positions.
- The x-axis is inverted to follow H-R diagram convention (hot left).
- Matplotlib FuncAnimation updates the star and trail at each frame.

The animation shows:
