Shock Front / Sedov–Taylor Blast Wave

Visualize 2D hydrodynamic shock propagation after a point explosion

Purpose

Demonstrate the propagation of a blast wave in 2D using the Sedov–Taylor solution, illustrating the growth of the shock front and density enhancement behind it.

Background

The Sedov–Taylor solution describes the self-similar propagation of a strong shock following an instantaneous point explosion in a uniform medium. It is widely used to model supernova explosions and blast waves in astrophysics.

Methodology

  1. Set up a 2D spatial grid.
  2. Compute radial distances from the explosion center.
  3. Compute shock radius as a function of time using Sedov–Taylor scaling.
  4. Compute density enhancement inside shock radius.
  5. Animate density field using Matplotlib’s imshow and FuncAnimation.

Pseudocode


Define 2D grid X, Y
Compute radial distance R from center
For each time step:
    Compute shock radius using Sedov–Taylor law
    Set density = high inside shock radius, low outside
    Update plot image
Animate frames using FuncAnimation

Python Simulation Code


import numpy as np
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation

plt.style.use("dark_background")


nx, ny = 100, 100
x = np.linspace(-1, 1, nx)
y = np.linspace(-1, 1, ny)
X, Y = np.meshgrid(x, y)
R = np.sqrt(X**2 + Y**2)


times = np.linspace(0.01, 2.0, 100)
E = 1.0    # explosion energy (arbitrary)
rho0 = 1.0 # ambient density

shock_radii = (E * times**2 / rho0)**0.2


def density_field(R, R_shock):
    """
    Simple approximation: inside shock radius -> higher density
    outside -> ambient density
    """
    rho = np.ones_like(R)
    rho[R <= R_shock] = 5.0  # shocked region density factor
    return rho

fig, ax = plt.subplots(figsize=(6,6))
im = ax.imshow(density_field(R, shock_radii[0]), extent=(-1,1,-1,1),
               origin='lower', cmap='plasma', vmin=1, vmax=5)
ax.set_title("Sedov–Taylor Blast Wave")
ax.set_xlabel("X")
ax.set_ylabel("Y")
plt.colorbar(im, ax=ax, label="Density")


def animate(i):
    rho = density_field(R, shock_radii[i])
    im.set_array(rho)
    return [im]

ani = FuncAnimation(fig, animate, frames=len(times), interval=50, blit=True)
plt.show()

Explanation

- The 2D grid represents a uniform medium.
- Density increases inside the shock front to visualize the blast.
- Shock radius grows with time as R ~ (E*t^2/ρ)^0.2.
- Animation shows the expanding shock front dynamically.

Expected Output

Animated 2D density map showing circular shock wave propagating outward. Colors represent density, with brighter regions indicating the shock front.
